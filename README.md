# MedVision – Breast Cancer Classification using Machine Learning

## Overview
MedVision is a machine learning classification project focused on predicting whether a breast tumor is benign or malignant using tabular medical data.

The goal of this project is to apply supervised machine learning techniques to a healthcare-related dataset and compare different classification models based on their performance.

## Objective
The main objective of this project is to build and evaluate machine learning models that can classify breast cancer cases as benign or malignant using medical features.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Joblib

## Machine Learning Models Used
- Logistic Regression
- Random Forest Classifier
- Support Vector Machine
- K-Nearest Neighbors

## Project Workflow
1. Imported the required Python libraries.
2. Loaded and explored the dataset.
3. Cleaned and preprocessed the data.
4. Handled missing values where required.
5. Encoded and scaled the features.
6. Split the dataset into training and testing sets.
7. Trained multiple machine learning models.
8. Evaluated the models using accuracy, confusion matrix, and ROC curve.
9. Compared model performances.
10. Saved the best-performing model using Joblib.

## Results
The Random Forest model achieved the best performance with an accuracy of approximately 90.7%.

Model comparison:
- Random Forest: 90.7%
- Logistic Regression: 86.0%
- KNN: 79.1%
- SVM: 60.5%
